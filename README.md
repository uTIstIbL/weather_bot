# Aaron 天氣機場資訊 Telegram Api

為了方便查詢當地機場的天氣資訊，特別開發了這款天氣的 API 供給大家做使用，裡面的功能目前不多
僅有查詢當地天氣資訊、Metar、以及 Taf，歡迎大家多多使用。

# 開始使用
* 下載檔案
```
git clone git@github.com:uTIstIbL/weather_bot.git
cd weather_bot
```

* 啟動伺服器
```
npm install
npm run start
or
nodemon index.js
```
